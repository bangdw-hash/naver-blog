ngrok이란?
ngrok은 로컬 PC의 앱을 인터넷에 공개해주는 무료 터널링 서비스입니다.

✅ 장점:
• 공유기 포트 포워딩 없이 즉시 사용
• HTTPS 자동 적용 (카메라 권한 등 정상 작동)
• 무료 계정으로도 충분 (세션당 하나의 터널)

⚠️ 주의:
• 앱이 실행 중일 때만 접속 가능
• 무료 계정은 앱 재시작마다 URL이 바뀜
• 고정 URL은 유료 플랜 또는 Railway 배포로 해결
접속이 안 될 때
🔴 WiFi 접속 — 연결 거부 오류
start.bat을 관리자 권한으로 실행했는지 확인하세요 (방화벽 자동 설정 포함)
🔴 WiFi 접속 — 페이지가 열리지 않음
Windows 방화벽 → 포트 5000 인바운드 허용 여부 확인
🔴 QR이 다른 IP를 가리킴
VPN이 켜져 있으면 끄고 새로고침 버튼을 누르세요
🔴 ngrok 오류: ERR_NGROK_108
인증 토큰이 없거나 잘못되었습니다. 설정 → 자동화 탭에서 토큰을 등록하세요
🔴 Railway 배포 시
Railway URL(https://xxx.up.railway.app)로 직접 접속하면 되므로 ngrok이 필요 없습니다
